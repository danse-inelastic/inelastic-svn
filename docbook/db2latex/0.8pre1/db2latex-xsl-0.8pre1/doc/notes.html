E-mail Services

A number of mailing lists are available. These are hosted by SourceForce.

Reporting Bugs or Requesting Features

Bugs and feature requests may be posted to the mailing lists or submitted to the SourceForce request tracker. A list of accumulated comments is also available.

DocBook Conformance

DB2LaTeX was developed with DocBook: The Definitive Guide 1st edition in mind. In time, it will be refined for the 2nd edition.

The DB2LaTeX Reference contains details of compliance, and links to DocBook: The Definitive Guide.
