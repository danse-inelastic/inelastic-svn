(callout) A. Description; B. Pertinent Variables.
Reference Section

Name

callout — Process callout [2.0.9] elements

Synopsis

<xsl:template match="callout"/><xsl:template match="callout" mode="generate.callout.arearef">
<xsl:param name="arearef" select="@arearefs"/>
<xsl:param name="area" select="key('id', $arearef)"/>
<xsl:param name="last" select="false()"/>
<xsl:param name="count" select="1"/>
<xsl:variable name="first" select="$count=1"/>
  ...
</xsl:template><xsl:template match="callout" mode="generate.callout.areasymbol"/>

A. Description

Formats arearefs as an \item, then applies templates. Since there may be multiple IDs specified in the arearefs attribute, the generate.callout.arearefs template is called recursively to generate the arearefs.

B. Pertinent Variables

There are no configuration variables for this template.

G. Variations

Variations on this template:

  • callout (in generate.callout.areasymbol mode)
