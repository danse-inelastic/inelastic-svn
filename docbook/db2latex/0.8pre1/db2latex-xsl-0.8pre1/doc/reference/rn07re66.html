D. DocBook Compliance and LaTeX Notes

        

          
Note

          

					This template does not apply to BibTeX bibliographies.
				

        

        

				A \bibitem is emitted.
				If can be linked via citation [2.0.9] or xref [2.0.9].
				The title [2.0.9] will be formatted in italics, followed by
				biblioentry.item.separator, the author [2.0.9]
				or authorgroup [2.0.9], and then the following elements separated
				by biblioentry.item.separator:
				copyright [2.0.9], publisher [2.0.9],
				pubdate [2.0.9], pagenums [2.0.9],
				isbn [2.0.9], editor [2.0.9],
				releaseinfo [2.0.9].
			

        

          
Note

          

					All templates for all DocBook elements will be applied
					with the “bibliography.mode” XSLT mode.
