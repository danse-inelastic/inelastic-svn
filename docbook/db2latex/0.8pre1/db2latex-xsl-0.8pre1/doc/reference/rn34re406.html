(latex.use.dcolumn) A. Description; F. Further Reading.
Reference Section

Name

latex.use.dcolumn — Toggle the use of the dcolumn LaTeX package

Synopsis

<xsl:param name="latex.use.dcolumn">0</xsl:param>

A. Description

Warning

Currently, dcolumn support does not function correctly.

dcolumn provides support for the char alignment of table cells.

F. Further Reading

  • latex.decimal.point

G. Uses

Documented templates that use this parameter:
