(latex.admonition.imagesize) A. Description.
Reference Section

Name

latex.admonition.imagesize — DB2LaTeX graphics admonitions size

Synopsis

<xsl:param name="latex.admonition.imagesize">width=1cm</xsl:param>

A. Description

Is passed as an optional parameter for \includegraphics and can take on any such legal values (or be empty).

G. Uses

Documented templates that use this parameter:

  • latex.admonition.environment
