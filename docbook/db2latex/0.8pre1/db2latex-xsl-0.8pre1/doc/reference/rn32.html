Revision History
Revision 1.1 May-January 2004  
Major review of documentation.

This file contains parameters that are shared with other XSL stylesheets such as those as http://docbook.sourceforge.net (see Parameter References in the DocBook XSL Stylesheet Reference Documentation). These are parameters are honoured so that you can coordinate your XHTML or FO stylesheets with DB2LaTeX.

Note

In some stylesheets, tests of parameter values use “!=0” logic. However, DB2LaTeX uses “=1” logic. This means that empty parameters are “off” in DB2LaTeX but “on” in those other stylesheets. The XPath values “true()” and “false()” work as expected.
