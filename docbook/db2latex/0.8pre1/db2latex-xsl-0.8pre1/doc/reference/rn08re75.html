Reference Section

Name

title|subtitle (in caption.mode mode) — Captions generated from title [2.0.9]s

Synopsis

<xsl:template match="title|subtitle" mode="caption.mode"/>

A. Description

Simply applies templates.

The formatting of titles in caption.mode may depend on the enclosing element's template.

B. Pertinent Variables

  • latex.table.caption.style

D. DocBook Compliance and LaTeX Notes

Since captions may be incorporated into hyperlinks and tables of cross references, “anchor”-type elements should not be applied when in this mode.
