Reference Section

Name

para|simpara — Process para [2.0.9] and simpara [2.0.9] elements

Synopsis

<xsl:template match="para|simpara"/>

A. Description

Starts new lines above and below its contents. Thus, consecutive paras [2.0.9] will have one blank line between them.

B. Pertinent Variables

  • latex.use.parskip

D. DocBook Compliance and LaTeX Notes

In LaTeX, there is no distinction between para [2.0.9] and simpara [2.0.9].

The accuracy of block elements within paras is unknown.

The use of para within footnotes [2.0.9] is unproven.
