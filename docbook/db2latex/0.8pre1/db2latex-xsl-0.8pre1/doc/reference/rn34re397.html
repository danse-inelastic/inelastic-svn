(latex.bridgehead.in.lot) A. Description.
Reference Section

Name

latex.bridgehead.in.lot — Control the inclusion of chapter titles in lots [2.0.9]

Synopsis

<xsl:param name="latex.bridgehead.in.lot">1</xsl:param>

A. Description

When this variable is set, lists of tables and lists of figures will be grouped and labeled by chapter.

G. Uses

Documented templates that use this parameter:

  • lot
  • generate.latex.common.preamble
