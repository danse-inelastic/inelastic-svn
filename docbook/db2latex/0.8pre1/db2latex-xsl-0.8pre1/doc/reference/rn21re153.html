(inlinegraphic) A. Description; B. Pertinent Variables.
Reference Section

Name

inlinegraphic — Process inlinegraphic [2.0.9] elements

Synopsis

<xsl:template match="inlinegraphic" name="inlinegraphic"/>

A. Description

Calls LaTeX \includegraphics with the value of fileref or the URI of entityref.

B. Pertinent Variables

There are no configuration variables for this template.
