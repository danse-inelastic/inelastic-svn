Name

author|editor|othercredit|personname — Process names of authors [2.0.9] and similar elements

Synopsis

<xsl:template match="author|editor|othercredit|personname"/>

A. Description

Formats a person's name for inline display.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

This template uses person.name from db2latex/xsl/common/common.xsl to format the names of author [2.0.9], editor [2.0.9], othercredit [2.0.9] and personname [2.0.9] elements.

For compatibility with LaTeX, normalize-scape is called on the output of person.name.list. This may pose problems but has not been investigated.
