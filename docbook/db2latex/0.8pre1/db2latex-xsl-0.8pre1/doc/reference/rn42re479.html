LaTeX Active character Escaping

(scape) A. Description; Mapping Source.
Reference Section

Name

scape — General text escaping for LaTeX active characters

Synopsis


      

      

        
        
A. Description

        

        Certain characters must be escaped so that LaTeX will typeset
        them verbatim rather than attempting to interpret them as commands.
