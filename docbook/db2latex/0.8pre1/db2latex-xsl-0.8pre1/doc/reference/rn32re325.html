(insert.xref.page.number) A. Description.
Reference Section

Name

insert.xref.page.number — Control the appearance of page numbers in cross references

Synopsis

<xsl:param name="insert.xref.page.number">0</xsl:param>

A. Description

When enabled, xrefs [2.0.9] will include page numbers after their generated cross-reference text.

G. Uses

Documented templates that use this parameter:

  • latex.use.varioref
  • xref|link
  • insert.label.markup (in insert.label.markup mode)
  • title (in insert.label.markup mode)
