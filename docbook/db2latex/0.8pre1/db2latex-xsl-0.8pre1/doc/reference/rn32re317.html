(author.othername.in.middle) A. Description.
Reference Section

Name

author.othername.in.middle — Control the display of othername [2.0.9] elements in authors [2.0.9]

Synopsis

<xsl:param name="author.othername.in.middle" select="1"/>

A. Description

When disabled, othername [2.0.9] elements will be suppressed when author [2.0.9] elements are formatted.
