Name

process.menuchoice — Process menuchoice [2.0.9] children (not shortcut)

Synopsis


      

      

        
        
A. Description

        

				Applies templates.
			

      

      

        
        
B. Pertinent Variables

        
There are no configuration variables for this template.

      

      

        
        
D. DocBook Compliance and LaTeX Notes

        

				Selects children of the current node (does not select shortcut [2.0.9] elements).
				Subsequent children are delimited by a plug sign, in general, or an arrow,
				for guimenuitem [2.0.9] and guisubmenu [2.0.9].
