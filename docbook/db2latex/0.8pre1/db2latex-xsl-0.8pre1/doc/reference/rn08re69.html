(blockquote) A. Description; F. Further Reading.
Reference Section

Name

blockquote — A quotation set off from the main text (not inline)

Synopsis

<xsl:template match="blockquote"/>

A. Description

Uses the LaTeX \begin{quote} environment. If an attribution is present, it will be set at the end.

F. Further Reading

  • attribution (in block.attribution mode)

  • The DB2LaTeX mapping system latex.mapping.xsl
