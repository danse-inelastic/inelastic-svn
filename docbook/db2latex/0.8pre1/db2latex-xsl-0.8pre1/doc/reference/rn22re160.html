Reference Section

Name

see|seealso (in indexterm mode) — Process the contents of see [2.0.9] and seealso [2.0.9] elements

Synopsis

<xsl:template match="see|seealso" mode="indexterm"/>

A. Description

Annotate an index entry with a “See” or “See also” cross-reference.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

Templates are applied. “See” or “see also” text is generated by gentext.element.name and formatted in italics.
