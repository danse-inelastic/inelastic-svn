Name

para/mediaobject — Process a para [2.0.9]'s mediaobject [2.0.9] elements

Synopsis

<xsl:template match="para/mediaobject"/>

A. Description

Formats a mediaobject [2.0.9] as a block surrounded by paragraph text.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

Inserts newline characters around the output of mediacontent.
