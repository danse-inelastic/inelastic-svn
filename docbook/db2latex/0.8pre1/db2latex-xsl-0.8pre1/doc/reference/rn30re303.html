Name

text() (in slash.hyphen mode) — Process text() nodes

Synopsis

<xsl:template match="text()" mode="slash.hyphen"/>

A. Description

Handles URL text content from DocBook documents.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

This template is only used by generate.string.url and only when latex.hyphenation.tttricks is disabled. LaTeX active characters will be escaped or hyphenated in a fashion that is tailored for URLs via scape.slash.hyphen.
