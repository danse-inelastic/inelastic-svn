(admon.graphic) A. Description; C. Template Parameters.
Reference Section

Name

admon.graphic — Choose an admonition graphic

Synopsis


      

      

        
        
A. Description

        


			For each admonition element
			(warning [2.0.9], tip [2.0.9], important [2.0.9], caution [2.0.9], note [2.0.9]),
			this template chooses the graphics filename. If the admonition element is
			not known, the note [2.0.9] graphic is used.

			

      

      

        
        
C. Template Parameters

        

          

            

              name
            

            

              


							The name of the adminition. DB2LaTeX includes
							different graphics for different adminitions. By
							default, name defaults to the
							XSLT local-name of the
							current node.
