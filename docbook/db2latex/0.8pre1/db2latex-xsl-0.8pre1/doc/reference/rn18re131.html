Name

generate.ulink.in.footnote — Process ulink [2.0.9] elements within footnote [2.0.9]s

Synopsis


      

      

        
        
A. Description

        

			Format a ulink [2.0.9].
			

      

      

        
        
B. Pertinent Variables

        
There are no configuration variables for this template.

      

      

        
        
D. DocBook Compliance and LaTeX Notes

        

				Footnotes are “tricky” and require special handling for
				a number of link-type commands.
			

        
This template exists in this file so that all
			the footnote-generating templates are close to each other. However,
			it is actually a part of the ulink template in xref.mod.xsl
