Reference Section

Name

refname — Process refname [2.0.9] elements

Synopsis

<xsl:template match="refname"/>

A. Description

Formats a refname [2.0.9] as a start-of-section for a refentry [2.0.9].

B. Pertinent Variables

  • refentry.generate.name

D. DocBook Compliance and LaTeX Notes

Will commence a LaTeX \section if necessary.

Designed to accommodate multiple refname [2.0.9]s in a single refentry [2.0.9].

The use of a comma between multiple refname [2.0.9]s should probably be localised.

  • The DB2LaTeX mapping system latex.mapping.xsl
