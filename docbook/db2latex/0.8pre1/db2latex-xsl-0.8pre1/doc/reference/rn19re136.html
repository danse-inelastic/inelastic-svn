Name

semiformal.object — Typeset formal or informal object generically

Synopsis


      

      

        
        
A. Description

        


				Chooses whether the current node is a “formal”-
				or “informal”-like object and calls either
				formal.object or
				informal.object.

			

      

      

        
        
B. Pertinent Variables

        
There are no configuration variables for this template.
