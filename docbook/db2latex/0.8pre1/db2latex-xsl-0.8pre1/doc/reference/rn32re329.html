(refentry.xref.manvolnum) A. Description.
Reference Section

Name

refentry.xref.manvolnum — Choose whether to include manvolnum [2.0.9] in cross-references

Synopsis

<xsl:param name="refentry.xref.manvolnum" select="1"/>

A. Description

When this option is enabled, manvolnums [2.0.9] will be displayed when cross-referencing refentries [2.0.9].

G. Uses

Documented templates that use this parameter:

  • manvolnum
