Name

biblioentry (in bibliography.all mode) — Process biblioentry [2.0.9] elements

Synopsis

<xsl:template match="biblioentry" mode="bibliography.all"/>

A. Description

Emits a bibiographic entry unconditionally.

F. Further Reading

This template calls biblioentry.output to format the entry.
