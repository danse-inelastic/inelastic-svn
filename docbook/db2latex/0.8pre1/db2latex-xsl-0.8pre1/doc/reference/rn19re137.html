Name

generate.formal.title.placement — Determine the relative position for a caption [2.0.9]

Synopsis


      

      

        
        
A. Description

        

				Chooses the position of a caption to be “before”
				or “after” the selected object.
			

      

      

        
        
B. Pertinent Variables

        

          
    
            
  • 
              formal.title.placement
            
    • 
          

        

      

      

        
        
C. Template Parameters

        

          

            

              object
            

            

              


						The type of object (i.e. name of the element) to which the caption pertains.

						

            

          

        

      

      

        
        
D. DocBook Compliance and LaTeX Notes

        


				If “after” has not been specified for the given
				object in formal.title.placement,
				“before” will be used by default.
