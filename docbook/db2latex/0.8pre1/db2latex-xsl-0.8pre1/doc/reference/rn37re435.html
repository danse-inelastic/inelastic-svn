Name

generate.latex.common.preamble — Common LaTeX preamble shared by articles [2.0.9] and books [2.0.9], and other document classes. Most of the packages and package options are managed here.

Synopsis


      

      

        
        
A. Description

        


			The LaTeX preamble, after the \documentclass command but before the
			\begin{document} environment.
