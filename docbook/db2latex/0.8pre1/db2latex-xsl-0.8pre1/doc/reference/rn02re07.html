Name

title (in latex mode) — Process title [2.0.9] elements

Synopsis

<xsl:template match="title" mode="latex"/>

A. Description

Applies templates with no mode. This template is called by extract.object.title.

This differs from the modeless title [2.0.9] templates because they suppress the output of titles.
