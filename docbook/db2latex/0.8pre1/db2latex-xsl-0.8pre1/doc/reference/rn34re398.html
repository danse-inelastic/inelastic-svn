(latex.fancyhdr.truncation.style) A. Description.
Reference Section

Name

latex.fancyhdr.truncation.style — Configure the application of truncation partitions

Synopsis

<xsl:param name="latex.fancyhdr.truncation.style">io</xsl:param>

A. Description

For latex.fancyhdr.truncation.partition, the partition can be modulated in a left-right fashion or an inside-outside fashion. Use “lr” for left-right, all other values are inside-outside (“io” is suggested). The default is “io”, because this matches latex.documentclass.book.

G. Uses

Documented templates that use this parameter:

  • latex.fancyhdr.truncation.partition
  • generate.latex.common.preamble
