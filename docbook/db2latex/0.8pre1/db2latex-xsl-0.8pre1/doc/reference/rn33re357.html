(latex.pagestyle) A. Description.
Reference Section

Name

latex.pagestyle — Override DB2LaTeX's choice of LaTeX page numbering style

Synopsis

<xsl:param name="latex.pagestyle"/>

A. Description

By default, DB2LaTeX will choose the 'plain' or 'fancy' page styles, depending on latex.use.fancyhdr. If non-empty, this variable overrides the automatic selection. An example would be the literal string 'empty', to eliminate headers and page numbers.

G. Uses

Documented templates that use this parameter:

  • generate.latex.pagestyle
