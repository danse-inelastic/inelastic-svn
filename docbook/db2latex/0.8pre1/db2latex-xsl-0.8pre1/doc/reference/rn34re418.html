(latex.alt.is.preferred) A. Description; F. Further Reading.
Reference Section

Name

latex.alt.is.preferred — Control the use of alt text

Synopsis

<xsl:param name="latex.alt.is.preferred">1</xsl:param>

A. Description

By default, DB2LaTeX assumes that alt text should be typeset in preference to any mediaobjects.

F. Further Reading

  • tex.math.in.alt

G. Uses

Documented templates that use this parameter:

  • equation
  • informalequation
  • inlineequation
