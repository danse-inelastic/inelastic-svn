Name

article/artheader|article/articleinfo (in standalone.article mode) — Process articleinfo [2.0.9] in article [2.0.9] elements

Synopsis

<xsl:template match="article/artheader|article/articleinfo" mode="standalone.article"/>

A. Description

Applies templates for legalnotice [2.0.9] and abstract [2.0.9].

B. Pertinent Variables

There are no configuration variables for this template.
