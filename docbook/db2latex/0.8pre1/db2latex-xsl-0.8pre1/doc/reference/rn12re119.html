(latex.preamble.essential.citation) A. Description; F. Further Reading.
Reference Section

Name

latex.preamble.essential.citation — Essential preamble for citation.mod.xsl support

Synopsis


      

      

        
        
A. Description

        


				Defines \docbooktolatexcite. This function
				helps to integrate bibioentry [2.0.9]/@id and
				bibioentry [2.0.9]/abbrev [2.0.9] with
				LaTeX.
				Also defines \docbooktolatexbackcite
				for compatability with hyperref's
				backref functionality.
