(latex.apply.title.templates) A. Description.
Reference Section

Name

latex.apply.title.templates — Whether to apply templates for component titles.

Synopsis

<xsl:param name="latex.apply.title.templates">1</xsl:param>

A. Description

Controls whether component titles will be generated by applying templates or by conversion to string values. When enabled, templates will be applied. This enables template expression in titles but may have problematic side-effects such as nested links.

Note

This variable does not influence all title elements. Some may have their own configuration variables or be non-configurable.

G. Uses

Documented templates that use this parameter:

  • extract.object.title
  • title|subtitle
  • procedure/title
