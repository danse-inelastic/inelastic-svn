1. Introduction

The DB2LaTeX mapping system centralises the mapping of DocBook tags (e.g. chapter [2.0.9]) to LaTeX commands (e.g. \chapter). It uses an auxiliary file, latex.mapping.xml, to define the start and end of each mapping. You can override this “mapping file” in order to customise DB2LaTeX.
