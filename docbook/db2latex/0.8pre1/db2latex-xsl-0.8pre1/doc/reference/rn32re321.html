(graphic.default.extension) A. Description.
Reference Section

Name

graphic.default.extension — Default filename extension for \includegraphics

Synopsis

<xsl:param name="graphic.default.extension"/>

A. Description

Specify the LaTeX search parameters for graphics filenames. If empty, DB2LaTeX will specify some explicit defaults.

G. Uses

Documented templates that use this parameter:

  • imagedata
  • latex.graphicext
