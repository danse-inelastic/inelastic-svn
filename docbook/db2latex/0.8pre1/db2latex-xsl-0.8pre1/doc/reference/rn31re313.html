Name

para-noline — Suppressed paragraphs

Synopsis

<xsl:template match="textobject/para|step/para|entry/para|question/para" name="para-noline"/>

A. Description

These paragraphs are not separated like normal paragraphs.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

This template exists to handle LaTeX problems with \par in certain contexts. These problems should be periodically reviewed by the DB2LaTeX team.
