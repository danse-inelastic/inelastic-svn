(latex.documentclass.article) A. Description.
Reference Section

Name

latex.documentclass.article — DB2LaTeX document class options for articles

Synopsis

<xsl:param name="latex.documentclass.article">a4paper,10pt,twoside,twocolumn</xsl:param>

A. Description

The article options are set to a4paper,10pt,twoside,twocolumn by default. These are the intermediate options to be passed to \documentclass, between the common options and the pdftex/dvips options.

G. Uses

Documented templates that use this parameter:

  • latex.article.preamblestart
