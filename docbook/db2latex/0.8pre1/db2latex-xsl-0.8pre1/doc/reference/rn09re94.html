Reference Section

Name

generate.latex.pagestyle — Choose the preferred page style for document body

Synopsis


      

      

        
        
A. Description

        


				If no page style is preferred by the user, the defaults will be
				empty for articles [2.0.9], plain
				for books [2.0.9], or
				fancy (if the LaTeX
				fancyhdr package is permitted).
