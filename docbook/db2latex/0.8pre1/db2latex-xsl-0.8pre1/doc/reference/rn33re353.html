(latex.procedure.title.style) A. Description.
Reference Section

Name

latex.procedure.title.style — Control the style of procedure titles

Synopsis

<xsl:param name="latex.procedure.title.style">\sc</xsl:param>

A. Description

Procedure titles are typeset in small caps but if this option is set to a LaTeX command, such as \textit, then that command will precede the title and it will be typeset accordingly.

G. Uses

Documented templates that use this parameter:

  • title|subtitle
  • title|subtitle (in caption.mode mode)
  • procedure/title
