(preferred.mediaobject.role) A. Description.
Reference Section

Name

preferred.mediaobject.role — Control mediaobject selection methods

Synopsis

<xsl:param name="preferred.mediaobject.role"/>

A. Description

When use.role.for.mediaobject is enabled, this variable can be used to specify the mediaobject [2.0.9] role that your document uses for LaTeX output. DB2LaTeX will try to use this role before using the “latex” or “tex” roles. For example, some authors may choose to set this to “pdf”.

G. Uses

Documented templates that use this parameter:

  • mediacontent
  • use.role.for.mediaobject
