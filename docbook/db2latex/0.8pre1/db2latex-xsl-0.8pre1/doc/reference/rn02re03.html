(latex.mapping.xml) A. Description; E. Samples and Tests.
Reference Section

Name

latex.mapping.xml — The primary mapping file

Synopsis

<xsl:param name="latex.mapping.xml" select="document('latex.mapping.xml')"/>

A. Description

DB2LaTeX will search for mappings in this file. The value of this variable must be an XML document. If mappings cannot be found in this file, DB2LaTeX will search the default mapping file.
