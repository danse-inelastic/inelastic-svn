Reference Section

Name

generate.simplelist.inline — Process simplelist [2.0.9] elements with inline members [2.0.9]

Synopsis

<xsl:template match="simplelist[@type='inline']" name="generate.simplelist.inline"/>

A. Description

Formats a simple, comma-separated list for a simplelist [2.0.9] that has a type attribute equal to “inline”.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

This is not made into a paragraph and is not temrinated by a full stop (“period”).

G. Variations

Variations on this template:
