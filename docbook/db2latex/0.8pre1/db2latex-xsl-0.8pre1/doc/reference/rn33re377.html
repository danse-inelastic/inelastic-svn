(latex.document.font) A. Description.
Reference Section

Name

latex.document.font — Document Font(s)

Synopsis

<xsl:param name="latex.document.font">palatino</xsl:param>

A. Description

Document fonts can be chosen by providing LaTeX package names. Common values include default, times, palatcm, charter, helvet, palatino, avant, newcent and bookman. Particular combinations may also work. For example, mathptm,charter,courier.

G. Uses

Documented templates that use this parameter:

  • generate.latex.common.preamble
