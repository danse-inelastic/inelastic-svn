Reference Section

Name

latex.use.parskip — Toggle the use of the parskip LaTeX package

Synopsis

<xsl:param name="latex.use.parskip">0</xsl:param>

A. Description

Use “block” paragraph style instead of indentation.

D. DocBook Compliance and LaTeX Notes

parskip introduces vertical whitespace between paragraphs and list items. However, DB2LaTeX's toc [2.0.9] and lot [2.0.9] templates attempt to suppress this whitespace.

When this option is off, you may wish to investigate latex.use.noindent.

F. Further Reading

G. Uses
