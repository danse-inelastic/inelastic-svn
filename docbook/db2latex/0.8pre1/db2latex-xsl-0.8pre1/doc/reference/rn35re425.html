Reference Section

Name

appendix — Process appendix [2.0.9] elements

Synopsis

<xsl:template match="appendix"/>

A. Description

Formats appendices.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

For each appendix, mappings are used and content templates are applies.

For appendix [2.0.9] elements that occur as children of book [2.0.9] or part [2.0.9], the LaTeX \chapter command will be used. Otherwise, the \section will be used.

If an appendix [2.0.9] is the first, or the last, then the LaTeX mappings for appendix-groups will be invoked in addition to the mapping for the appendix itself. For chapter-level appendices, the “appendices-chapter” is used. Otherwise, the “appendices-section” mapping is used.
