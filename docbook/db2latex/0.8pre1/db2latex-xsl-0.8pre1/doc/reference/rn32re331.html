(rootid) A. Description.
Reference Section

Name

rootid — Process only one element tree within a document

Synopsis

<xsl:param name="rootid" select="''"/>

A. Description

When this variable is non-empty, it is interpreted as the ID of an element that should be typeset by DB2LaTeX. The element's children, but none of its siblings or ancestors, will be processed as per normal. When the root element is a book [2.0.9] or article [2.0.9], that component will have its normal infrastructure (including bookinfo [2.0.9] or articleinfo [2.0.9]) processed before the “rootid” element.
