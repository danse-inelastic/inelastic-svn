Name

authorinitials — Process authorinitials [2.0.9] elements

Synopsis

<xsl:template match="authorinitials"/>

A. Description

Represents authorinitials [2.0.9] by applying templates normally and then appending biblioentry.item.separator.

B. Pertinent Variables

  • biblioentry.item.separator

D. DocBook Compliance and LaTeX Notes

The use of biblioentry.item.separator should be replaced with the normal localisation mechanism.
