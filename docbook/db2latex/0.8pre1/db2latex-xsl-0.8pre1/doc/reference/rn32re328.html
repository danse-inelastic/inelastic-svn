(use.role.as.xrefstyle) A. Description.
Reference Section

Name

use.role.as.xrefstyle — Honour role as proxy for xrefstyle

Synopsis

<xsl:param name="use.role.as.xrefstyle">0</xsl:param>

A. Description

The xrefstyle attribute is not yet part of DocBook so the role attribute can be used until xrefstyle is available for xref [2.0.9] elements.

G. Uses

Documented templates that use this parameter:

  • generate.xref.text
