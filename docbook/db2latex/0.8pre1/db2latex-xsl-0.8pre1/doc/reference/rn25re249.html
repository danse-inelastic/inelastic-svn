Reference Section

Name

generate.label.id — Generate a reliable cross-reference id

Synopsis


      

      

        
        
A. Description

        

			
			The passed argument indicates the object whose id attribute is used to generate the
			label. In this sense, in most cases its the current node itself. If
			the used object has not an id attribute, a unique id is obtained by
			means of the generate-id function. Moreover, if
			we are using the hyperref package, a hypertarget is also defined
			for this object.
			
			

      

      

        
        
B. Pertinent Variables

        
There are no configuration variables for this template.

      

      

        
        
C. Template Parameters

        

          

            

              object
            

            

              
The node whose id is to be used.
