(book/bookinfo) A. Description; D. DocBook Compliance and LaTeX Notes.
Reference Section

Name

book/bookinfo — Process bookinfo [2.0.9] elements

Synopsis

<xsl:template match="book/bookinfo"/>

A. Description

Processes a book [2.0.9]'s bookinfo [2.0.9] (will be invoked after the title page has been typeset).

D. DocBook Compliance and LaTeX Notes

Only the revhistory [2.0.9], abstract [2.0.9], keywordset [2.0.9], copyright [2.0.9] and legalnotice [2.0.9] are processed. Users may override this in their customisation layer.
