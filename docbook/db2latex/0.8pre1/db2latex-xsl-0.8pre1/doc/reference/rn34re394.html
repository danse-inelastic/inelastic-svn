(latex.is.draft) A. Description; E. Samples and Tests.
Reference Section

Name

latex.is.draft — Toggle the use of the the “draft” preamble

Synopsis

<xsl:param name="latex.is.draft"/>

A. Description

DB2LaTeX provides a number of draft-mode features to aid the refinement of documents. Normally, this mode is enabled or disabled according to the status attribute of the top-level book [2.0.9] or article [2.0.9] element. However, this variable will take precedence when it is not empty. It is empty by default.

E. Samples and Tests

test_index_draft.pdf from test_index/test.xml using draft.xsl, test_draft.pdf from test_draft/test.xml

G. Uses

Documented templates that use this parameter:

  • indexterm
  • show.comments
  • generate.latex.common.preamble
  • generate.latex.draft.preamble
