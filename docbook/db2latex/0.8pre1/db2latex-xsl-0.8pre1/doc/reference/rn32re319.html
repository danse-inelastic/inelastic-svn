(toc.section.depth) A. Description.
Reference Section

Name

toc.section.depth — Cull table-of-contents entries that are deeply nested

Synopsis

<xsl:param name="toc.section.depth">4</xsl:param>

A. Description

Specifies the maximum depth before sections are omitted from the table of contents.

G. Uses

Documented templates that use this parameter:

  • article
