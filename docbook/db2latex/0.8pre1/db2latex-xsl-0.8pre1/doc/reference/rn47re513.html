Name

verbatim.apply.templates — Auxiliary template to output verbatim LaTeX code in verbatim mode

Synopsis


      

      

        
        
A. Description

        
Typeset verbatim, monospace text in which whitespace is significant.

      

      

        
        
B. Pertinent Variables

D. DocBook Compliance and LaTeX Notes

        
This template is called by a number of element-related templates.

        
 Takes into account whether the user is using fancyvrb or not. It allows
			veratim line numbering and other fancy stuff. 

        
 In order to use a small or large font, you may also want to use 
			implement latex.fancyvrb.options.
			

        

					This template will apply further templates using the
					latex.verbatim mode.
				

        

          
Warning

          

					This doesn't work inside tables [2.0.9].
					Also, if the element appears within a varlistentry [2.0.9],
					some LaTeX code will be emitted so that the verbatim environment
					starts on a new line.
