Name

latex|tex — Process latex and tex elements

Synopsis

<xsl:template match="latex|tex"/>

A. Description

Passes contents through as raw LaTeX text.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

The latex and tex elements are not part of DocBook.
