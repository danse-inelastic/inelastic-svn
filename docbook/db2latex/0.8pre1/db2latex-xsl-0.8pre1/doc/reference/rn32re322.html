(use.role.for.mediaobject) A. Description.
Reference Section

Name

use.role.for.mediaobject — Control mediaobject selection methods

Synopsis

<xsl:param name="use.role.for.mediaobject">1</xsl:param>

A. Description

This controls how DB2LaTeX behaves when a figure [2.0.9] contains multiple mediaobjects [2.0.9]. When enabled, DB2LaTeX will prefer the mediaobject with the “latex”, “tex” or preferred.mediaobject.role role, if any.

G. Uses

Documented templates that use this parameter:

  • mediacontent
  • preferred.mediaobject.role
