(latex.table.caption.style) A. Description.
Reference Section

Name

latex.table.caption.style — Control the style of table captions

Synopsis

<xsl:param name="latex.table.caption.style"/>

A. Description

Figure captions are typeset in the default typeface (usually 'roman') but if this option is set to a LaTeX command, such as \textit, then that command will precede the caption and it will be typeset accordingly.

G. Uses

Documented templates that use this parameter:

  • title|subtitle (in caption.mode mode)
  • table
