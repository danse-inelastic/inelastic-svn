Name

footnote — Process footnote [2.0.9] elements

Synopsis

<xsl:template match="footnote"/>

A. Description

Format a footnote.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

Applies templates within a LaTeX \footnote command. Note that this may not work within some tables. Also, indexterms [2.0.9] may fail.

This template requires LaTeX commands that are part of the “essential preamble”.
