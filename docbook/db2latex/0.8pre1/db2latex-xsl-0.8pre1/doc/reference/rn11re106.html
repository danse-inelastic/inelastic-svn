Reference Section

Name

area (in generate.arearef mode) — Illustrate a reference to a callout's area

Synopsis

<xsl:template match="area" mode="generate.arearef">
<xsl:param name="units"/>
  ...
</xsl:template>

A. Description

Formats an area [2.0.9] as part of a callout list.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

This is a fallback template for unknown units. It does not format an arearef but instead prints an error message.

G. Variations

Variations on this template:
