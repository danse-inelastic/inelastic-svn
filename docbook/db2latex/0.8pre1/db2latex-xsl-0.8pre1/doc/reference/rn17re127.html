Name

generate.aux.informalfigure.caption — Typeset a caption for an informal figure

Synopsis


      

      

        
        
A. Description

        

			
			Formats a caption, if any, as a centred block.
			
			

      

      

        
        
B. Pertinent Variables

        
There are no configuration variables for this template.

      

      

        
        
D. DocBook Compliance and LaTeX Notes

        


				A caption, if any, is gleaned from the caption [2.0.9]
				child of any mediaobject [2.0.9] contained within the
				informalfigure [2.0.9], and will be formatted as a
				centred block.
