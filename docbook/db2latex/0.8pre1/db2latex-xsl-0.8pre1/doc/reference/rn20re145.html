(glossentry/abbrev) A. Description; B. Pertinent Variables; D. DocBook Compliance and LaTeX Notes.
Reference Section

Name

glossentry/abbrev — Process a glossentry [2.0.9]'s abbrev [2.0.9] elements

Synopsis

<xsl:template match="glossentry/abbrev"/>

A. Description

Format an abbrev as part of a glossentry.

B. Pertinent Variables

There are no configuration variables for this template.

D. DocBook Compliance and LaTeX Notes

The abbrev [2.0.9] is delimited by square brackets. It appears in the “body” (e.g. glossdef [2.0.9]) region of the glossary entry, not as part of the glossterm [2.0.9].

The delimiters should probably be localised.
