(content templates) A. Description; C. Template Parameters.
Reference Section

Name

content-templates — Generic handler for interior elements of block-formatted components

Synopsis


      

      

        
        
A. Description

        

			Applies all templates, excluding from title [2.0.9]-related
			and blockinfo [2.0.9]-like elements.
			

      

      

        
        
C. Template Parameters

        

          

            

              info
            

            

              


						The name of the DocBook “info”-type
						element for this component. By default, this is equal
						to the name of the component with info
						appended. For example: sectioninfo [2.0.9]
						for section [2.0.9].
