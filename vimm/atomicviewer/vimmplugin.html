Vimm: Visual Interface to Materials Manipulation

This is a simple demo of the more powerful version in VNF: Virtual Neutron Facility.

Rotate
click and drag next to the atomic structure
Zoom
scroll (note: may not work for mac users)
Change structure
click on combo box on the side
This uses the plugin version of o3d. To see webgl Vimm with the latest webgl version of o3d, you must have a webgl-enabled browser.
