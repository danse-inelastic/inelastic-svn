1) Introduction / Purpose

   Cassandra is a modular and advanced 3D scientific data viewer.
   Cassandra is written in Java and based on the Visualisation Tool Kit, 
   VTK. 

   The Cassandra pipeline manager is an implementation of the Cassandra 
   PipelineManager that take care of the VTK object management.

2) Install and use

   To install Cassandra PipelineManager just link it with 
   the CassandraFramework. For that update the CassandraFramework
   ClassPath.
